1. **Clone the repository**:
   ```bash
   git clone https://github.com/your-username/TypoGraph.git
   ```

   ```bash
   npm install
   ```
3. **Run the development server**:
   ```bash

   ```
4. **Open your browser** and navigate to `http://localhost:5173` to see the app in action.

